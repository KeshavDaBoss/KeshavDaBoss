# Projects

My project work is currently centred around one flagship system, SmartPark, as well as Project_Numidia, my next moderately big project, supported by a few smaller exploratory builds that served primarily as learning and experimentation phases.

SmartPark is an end-to-end intelligent parking management system designed to address real-world congestion and inefficiencies in large public parking spaces such as malls. The system uses a Raspberry Pi and an ESP32 to collect real-time data from multiple sensors deployed across different parking zones, which is then processed, stored, and visualised through a structured backend. Though it should be mentioned that it is being actively upgraded. The project focuses on practical system design, including hardware integration, data flow, database structuring, and real-time status updates, rather than theoretical simulation.  


Alongside SmartPark, I have worked on a small number of exploratory projects focused on testing specific ideas or technologies, many made for hackathons. These builds were intentionally limited in scope and were used to gain hands-on familiarity with hardware integration, logic flow, and rapid prototyping. While not flagship efforts, they contributed to the technical foundation that enabled the execution of SmartPark at a larger and more structured level.
